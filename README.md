# OS-Override
这是SCU 2023Minigamed的课程项目，在腾讯和网易的老师们指导下完成。

游戏介绍：
在《OS Override》中用APP抵御病毒！下载各式各样的APP，使用它们的特殊能力！随着时间流逝，敌人的进攻愈发猛烈！合理安排你手机中的资源，你可以坚持多久？
